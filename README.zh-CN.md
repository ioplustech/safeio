# safeio

safeio 是一个用于轻松管理 Safe 多签钱包的 npm 包，。

## 安装

```bash
npm install safeio
```

## 使用方法

### 作为 npm 包

```javascript
const safeio = require('safeio');

// 使用 safeio 函数
```

## 功能

- 创建和管理 Safe 多签钱包
- 执行交易
- 查看钱包信息和交易历史
- 轻松集成到现有项目中

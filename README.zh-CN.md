# safeio

safeio 是一个简化的命令行工具和 npm 包，用于轻松管理 Safe 多签钱包。

## 安装

```bash
npm install -g safeio
```

## 使用方法

### 作为命令行工具

```bash
safeio [命令] [选项]
```

### 作为 npm 包

```javascript
const safeio = require('safeio');

// 使用 safeio 函数
```

## 功能

- 创建和管理 Safe 多签钱包
- 执行交易
- 查看钱包信息和交易历史
- 轻松集成到现有项目中
